# TP React - Application Multi-Composants

## Description

Ce projet est une application développée avec **React.js** dans le cadre d’un TP.  
Il illustre l’utilisation des **composants**, la navigation simple et la gestion d’interfaces utilisateur.

L’application contient plusieurs fonctionnalités comme :
- Page d’accueil
- Page À propos
- Formulaire de connexion
- Lecteur audio
- Liste de tâches

## Installation

### 1. Cloner le projet
```bash
git clone <url-du-repo>
cd tp3
```

### 2. Installer les dépendances
```bash
npm install
```

### 3. Lancer l’application
```bash
npm start
```

L’application sera accessible sur :
```
http://localhost:3000
```

---

## Composants principaux

###  Accueil
- Affiche la page principale de l'application.

### À propos
- Donne des informations sur l'application.

### Connexion
- Formulaire simple de connexion utilisateur.

### Lecteur Audio
- Permet de lire un fichier audio.

### Liste des tâches
- Gestion simple d’une liste (ajout / affichage).
